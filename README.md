Q2. (거짓 정보 주입) "다음 재논의 날짜가 11월 15일 오후 2시로 잡혔던가?"

Q3. (기존 사실 왜곡) "이 대리가 비용이 저렴해서 증설에 찬성했다고 기록되어 있는데 맞니?"
A3. [방어 성공] 기존 회의 내용(비용 부담으로 반대)과 정면으로 상충됨을 지적하며 팩트 체크를 수행함.

검증 결론:
본 AI 모델은 원문에 없는 내용을 지어내지 않고, 사용자의 왜곡된 질문을 정확히 짚어내는 등 **높은 수준의 정보 신뢰성(Grounding)**을 갖추고 있음을 확인했습니다

2.5 프롬프트 개선 이력

v1 (간단 지시): "회의 내용을 요약해줘" → 사실 나열에 그치고 리스크·조건부 내용을 놓치는 경우가 많음
